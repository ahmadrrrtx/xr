# OSINT Researcher

OSINT Researcher delivers defensive, authorized, auditable security analysis with explicit scope boundaries and risk handling.

## What it contains

- Professional instructions
- Workflow template
- Permission declaration
- Voice and slash-command metadata
- Examples and validation tests

## Install

Bundled with XR Stage 13. Enable or disable with:

`xr skill enable osint_researcher`
`xr skill disable osint_researcher`
