# XR 3.1G — 3D Strategy

**Date:** 2026-07-14

## Mandate

Build a **real** 3D experience using Three.js / React Three Fiber. No fake CSS transforms.

## 3D Concepts (prioritized)

1. **XR Core** — Floating neural orb in space (Hero)
2. **Neural Network Visualization** — Live agent connections
3. **Knowledge Graph** — Memory engine
4. **Orbiting AI Nodes** — Multi-agent orchestration
5. **Dynamic Data Particles** — Research engine
6. **Holographic Architecture** — Interactive system map
7. **Live System Visualization** — Computer control
8. **Multi-agent Orchestration** — Real-time demo
9. **Memory Network** — Timeline + connections

## Technical Stack

- React Three Fiber
- @react-three/drei
- Three.js
- Post-processing (subtle bloom, depth of field)

## Performance Rules

- Max 60fps target
- LOD (level of detail) where needed
- Pause when not in viewport
- Mobile fallback to 2D or simplified 3D

## Purpose

Every 3D element must communicate a core XR concept. Never decorative.
