# XR 3.1G — Visual Language

**Date:** 2026-07-14

## Core Aesthetic

**Minimal. Premium. Elegant. Futuristic. Professional. Calm. High contrast. Breathing whitespace. Large typography. 3D depth. Soft motion. Zero clutter.**

## Influences (in order of priority)

1. Apple — Ultimate product storytelling
2. Linear — Physical motion & clarity
3. Nothing — Futuristic minimalism
4. Vercel — Developer elegance
5. Raycast — Speed and calm
6. Figma — Thoughtful systems thinking

## Typography

- Primary: Inter / SF Pro Display (system-native)
- Headings: Large, confident, generous tracking
- Body: Excellent readability at all sizes
- Monospace: For CLI, code, and technical content

## Color Palette (Official XR)

- Primary: Deep space black (#0A0A0C)
- Accent: Electric blue (#3B82F6) — from official assets
- Secondary: Soft white (#F8FAFC)
- Neutrals: Slate-900 to Slate-100
- Subtle gradients only when purposeful

## Logo & Avatar Usage

Official assets from `/assets/logo.png` and `/assets/avatar.png` must be used exclusively. No recreation.

## 3D & Depth

Real Three.js / React Three Fiber with:
- Real lighting
- Camera controls where appropriate
- Subtle particle systems
- Holographic / neural aesthetics

## Do Not Use

- Glassmorphism spam
- Neon overload
- Random gradients
- Heavy shadows
- Cluttered layouts
- SaaS templates
