# XR 3.1G — Competitive Research

**Date:** 2026-07-14

## 1. AI Category Leaders

**OpenAI** — Masterful use of breathing whitespace, bold typography, and calm confidence. Product feels inevitable.

**Claude** — Clean, thoughtful, high-trust. Anthropic’s site feels like a serious operating system for thought.

**Cursor** — Best-in-class developer positioning. Emphasizes "the best way to code" with subtle motion and excellent demos.

**Windsurf** — High-quality 3D + motion showing agent orchestration.

**Perplexity** — Research-focused clarity. Excellent use of real-time data.

**Lovable** — Beautiful, emotional design. Very high production value.

**Vercel AI** — Extremely polished. Perfect example of developer-first futuristic design.

**Continue.dev** — Open-source ethos done right. Clean, community-first.

**OpenHands / Goose / Devin / OpenCode** — Strong agent positioning but often feel like "tools" rather than platforms.

## 2. Developer Infrastructure

**Vercel** — Gold standard for developer websites. Typography, motion, and developer experience are unmatched.

**Supabase / Railway / Fly.io / Cloudflare** — Excellent use of real-time dashboards and calm technical aesthetics.

**Linear** — Best motion system in the industry. Physical, spring-based, delightful.

**Raycast** — Command-palette elegance translated to web. Extremely fast and calm.

**GitHub** — Authority and trust through simplicity.

## 3. Design Systems

**Apple** — Ultimate reference for premium product storytelling.

**Nothing** — Futuristic minimalism with personality.

**Arc Browser** — Innovative interface metaphors.

**Framer / Figma / Stripe / Notion** — Each demonstrates different strengths in motion, typography, and clarity.

## 4. Futuristic Reference Analysis

**Primary Reference:** https://v0-compute-11.vercel.app/

**Key learnings:**
- Deep 3D with real Three.js lighting and camera work
- Extremely restrained color palette (blacks, deep blues, soft whites)
- Large, confident typography
- Physical-feeling motion
- Data visualizations that feel alive but not noisy
- Perfect balance of atmosphere and clarity
