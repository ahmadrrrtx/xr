# XR 3.1G — Mobile Strategy

**Date:** 2026-07-14

## Philosophy

Mobile experience must feel premium — not a degraded version.

## Approach

- Fully responsive from 320px
- 3D sections gracefully degrade (static or simplified)
- Touch-friendly interactions
- Same content hierarchy
- Performance parity with desktop

## Breakpoints

- Mobile: < 768px
- Tablet: 768–1024px
- Desktop: > 1024px

Mobile-first development with desktop enhancements.
