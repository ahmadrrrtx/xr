# XR 3.1G — Documentation Strategy

**Date:** 2026-07-14

## Approach

Documentation is **not** a separate docs site. It is beautifully integrated into the website.

## Features

- Interactive CLI & Shell previews
- Searchable capability explorer
- Embedded code examples
- One-click copy for install commands
- Links to full GitHub docs when deeper reading needed

## Goal

Users should feel they are exploring the actual product, not reading marketing copy.
