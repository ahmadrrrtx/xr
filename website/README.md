# XR Website

The official production website for XR — the agentic runtime for software.

Built with Next.js 16 (App Router), React 19, Tailwind CSS v4, and Framer Motion.

## Develop

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
npm start
```

## Deploy

Deploys cleanly to Vercel with zero configuration.

## Structure

- `src/app/` — App Router pages (one per public route)
- `src/components/` — UI primitives and shared components
- `src/lib/` — Site config, data, and utilities
- `public/` — Static assets
