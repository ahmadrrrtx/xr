Core Capabilities

JARVIS Computer Control
XR sees your screen via screenshots, reasons about what's on it, and takes actions — click, type, scroll, open apps — just like you would. The agent isn't just answering questions. It's operating the computer.
NEW — XR Only
💰
Cost Governor (Enforced)
A hard, deterministic spend/token ceiling enforced in code. The agent PHYSICALLY cannot exceed it — checkBeforeStep() runs before every model call and blocks if the next step would breach your budget.
Headline Differentiator
🛡️
Provable Security
xr test --attacks --json runs an injection corpus and prints a SHA-256-signed, publishable block-rate. You measure security instead of marketing it. No blockchain needed.
Industry First
🔗
Tamper-Evident Audit Log
SHA-256 hash chain in SQLite (git's trick). $0 to run, offline, private. Every action is logged and verifiable. API keys are redacted before storage.
Zero Infrastructure
🧠
Non-Regressive Skills
Verified-good action sequences are frozen as immutable baselines. Any skill update that breaks a past win is auto-rolled-back. The agent cannot forget what worked.
No Catastrophic Forgetting
🔄
Self-Improving (Hermes-Style)
After every successful task, XR analyzes what it did. If verifiable and complex enough, it creates a skill that self-improves during use. Cross-session memory via FTS5 search.
Learns From Experience
🐳
Docker Sandbox
Shell commands run in isolated containers with dropped capabilities, no network, and memory limits. rm -rf / and curl | bash attacks are structurally impossible.
Zero Privilege Escalation
🎙️
Voice Control
Wake word → Whisper STT → XR agent → Kokoro TTS. Local by default. "Hey XR, open Safari and check my calendar." Full voice pipeline, zero cloud dependency by default.
Private by Design
🖥️

Security Model — Honest

Blast-radius reduction you can measure.

🚪

Egress Allow-List

The agent can't reach a domain you didn't approve. Exfiltration structurally impossible.

✋

Approval Gates

Write / delete / shell need explicit approval. Fail-closed on timeout (5 min default).

🪪

Tamper-Evident Audit

SHA-256 hash chain in SQLite. Verify offline, no blockchain needed. Detects any change.

🐳

Docker Sandbox

Shell commands in isolated container. Dropped capabilities, no network, memory-capped.

🧪

Injection Test Lab

Publishable block-rate with SHA-256 signature. We measure security, don't market it.

🔑

BYOK — Zero Secrets

XR ships no API keys, stores none. Keys come from your environment / OS keychain only.
