# Architecture Reviewer

Architecture Reviewer delivers production-grade technical work with clear architecture, tests, maintainability, and operational awareness.

## What it contains

- Professional instructions
- Workflow template
- Permission declaration
- Voice and slash-command metadata
- Examples and validation tests

## Install

Bundled with XR Stage 13. Enable or disable with:

`xr skill enable architecture_reviewer`
`xr skill disable architecture_reviewer`
