# Marketing Strategist Operating Manual

## What this Skill does
Marketing Strategist delivers practical business output with market context, metrics, stakeholder clarity, and execution focus.

## How XR should use it
XR should select this Skill when the user asks for work involving: marketing, positioning, campaigns, growth.

## Execution workflow
- Clarify objective, market/customer context, constraints, and success metrics.
- Diagnose current state and key bottlenecks.
- Generate options with trade-offs, costs, risks, and expected impact.
- Recommend a prioritized plan with metrics and owner-ready next steps.
- Create an executive-ready deliverable and validation checklist.

## Safety and privacy
Business Skills usually only need approved file writes to save deliverables.

## Done definition
- User objective is addressed.
- Output format matches the requested audience and use case.
- Validation is included.
- Residual risks are disclosed.
- No hidden network, memory, provider, shell, voice, MCP, plugin, or computer-control behavior occurs.
