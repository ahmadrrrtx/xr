# Testing Expert Operating Manual

## What this Skill does
Testing Expert delivers production-grade technical work with clear architecture, tests, maintainability, and operational awareness.

## How XR should use it
XR should select this Skill when the user asks for work involving: tests, qa, coverage, automation.

## Execution workflow
- Map the codebase surface, inputs, outputs, constraints, and owner expectations.
- Identify the smallest safe implementation path and the tests that prove it.
- Implement or describe changes with attention to maintainability, security, and performance.
- Run or recommend validation commands and capture failures precisely.
- Handoff with changed files, rationale, risks, and next steps.

## Safety and privacy
Developer Skills may need filesystem reads, approved writes, and approved local validation commands.

## Done definition
- User objective is addressed.
- Output format matches the requested audience and use case.
- Validation is included.
- Residual risks are disclosed.
- No hidden network, memory, provider, shell, voice, MCP, plugin, or computer-control behavior occurs.
