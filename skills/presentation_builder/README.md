# Presentation Builder

Presentation Builder delivers distinctive creative work grounded in audience, constraints, taste, iteration, and execution quality.

## What it contains

- Professional instructions
- Workflow template
- Permission declaration
- Voice and slash-command metadata
- Examples and validation tests

## Install

Bundled with XR Stage 13. Enable or disable with:

`xr skill enable presentation_builder`
`xr skill disable presentation_builder`
