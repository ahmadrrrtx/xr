# UX Designer Operating Manual

## What this Skill does
UX Designer delivers distinctive creative work grounded in audience, constraints, taste, iteration, and execution quality.

## How XR should use it
XR should select this Skill when the user asks for work involving: ux, research, flows, usability.

## Execution workflow
- Define audience, goal, brand/taste direction, constraints, and deliverable format.
- Explore creative territories and select a coherent direction.
- Produce polished draft artifacts with rationale.
- Critique against audience, clarity, originality, and conversion/usability.
- Deliver final version plus variants, usage guidance, and next iteration ideas.

## Safety and privacy
Creative Skills may need approved writes to save copy, design briefs, or presentation artifacts.

## Done definition
- User objective is addressed.
- Output format matches the requested audience and use case.
- Validation is included.
- Residual risks are disclosed.
- No hidden network, memory, provider, shell, voice, MCP, plugin, or computer-control behavior occurs.
