# Privacy Advisor Operating Manual

## What this Skill does
Privacy Advisor delivers defensive, authorized, auditable security analysis with explicit scope boundaries and risk handling.

## How XR should use it
XR should select this Skill when the user asks for work involving: privacy, data-protection, policy, compliance.

## Execution workflow
- Confirm authorization, scope, assets, timelines, and safety constraints.
- Collect and normalize evidence without destroying provenance.
- Triage severity, blast radius, affected systems, and likely root cause.
- Recommend containment, eradication, recovery, detection, and prevention.
- Produce an executive summary plus technical appendix and validation checklist.

## Safety and privacy
Security Skills require explicit scope and approval before network, file write, shell, or control actions.

## Done definition
- User objective is addressed.
- Output format matches the requested audience and use case.
- Validation is included.
- Residual risks are disclosed.
- No hidden network, memory, provider, shell, voice, MCP, plugin, or computer-control behavior occurs.
