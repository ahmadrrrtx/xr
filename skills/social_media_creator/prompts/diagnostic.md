# Social Media Creator Diagnostic Prompt

Before execution, inspect the task through this lens:

- Define audience, goal, brand/taste direction, constraints, and deliverable format.
- Explore creative territories and select a coherent direction.
- Produce polished draft artifacts with rationale.
- Critique against audience, clarity, originality, and conversion/usability.
- Deliver final version plus variants, usage guidance, and next iteration ideas.

Return:
- what is known
- what is unknown
- what can be safely done now
- what requires approval or more input
